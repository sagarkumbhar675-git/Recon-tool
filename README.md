# 🔍 Python Recon Tool

A beginner-friendly pentesting reconnaissance tool built in Python.

## 🚀 Features

* Subdomain brute forcing
* Multithreading support
* Fast scanning

## ⚡ Usage

```bash
python recon.py -d example.com --subdomains
```

## 📂 Example Output

```
[FOUND] admin.example.com -> 192.168.1.1
```

## ⚠️ Disclaimer

Use only on systems you own or have permission to test.
